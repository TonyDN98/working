# Expressions and statements

## Arithmetic Operators

PEMDAS
---

```
() -> Parentheses
** -> Exponents
*  -> Multiplication
/  -> Floating Division Returns 3.5
// -> Floar Division Return rounded to lower number
%  -> Modulo
+  -> Addition
-  -> Subtraction
```

---

```python
>>> 7 / 4
1.75
>>> (2 + 6) / 4
2.0
>>> 7 // 4        # Floor division (rounding down)
1
>>> 7 % 4         # Modulus (remainder of 7 // 4)
3

>>> x = 20
>>> x + 2
```
